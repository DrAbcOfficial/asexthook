    BeamFlags

    Beam flags used by CBeam::SetFlags

    Values

    Name Value Description
    BEAM_FSINE 16 Beam shape follows a sinus shape
    BEAM_FSOLID 32 Beam is a solid shape
    BEAM_FSHADEIN 64 Beam fades in at its start point
    BEAM_FSHADEOUT 128 Beam fades out at its end point
