CLASS

Entity classification
Can be set on entities with KeyValue using key 'classify'.

Values

Name Value Description
CLASS_FORCE_NONE -1 Forces class override to none
CLASS_NONE 0 No classification
CLASS_MACHINE 1 Is machine
CLASS_PLAYER 2 Is player
CLASS_HUMAN_PASSIVE 3 Is passive human
CLASS_HUMAN_MILITARY 4 Is military human
CLASS_ALIEN_MILITARY 5 Is military alien
CLASS_ALIEN_PASSIVE 6 Is passive alien
CLASS_ALIEN_MONSTER 7 Is monster alien
CLASS_ALIEN_PREY 8 Is prey alien
CLASS_ALIEN_PREDATOR 9 Is predator alien
CLASS_INSECT 10 Is insect
CLASS_PLAYER_ALLY 11 is player ally
CLASS_PLAYER_BIOWEAPON 12 Is player bioweapon
CLASS_ALIEN_BIOWEAPON 13 Is alien bioweapon
CLASS_XRACE_PITDRONE 14 Is X-Race pitdrone
CLASS_XRACE_SHOCK 15 Is X-Race shock trooper (also Voltigores)
CLASS_TEAM1 16 Is on Team 1
CLASS_TEAM2 17 Is on Team 2
CLASS_TEAM3 18 Is on Team 3
CLASS_TEAM4 19 Is on Team 4
CLASS_BARNACLE 99 Is Barnacle
