    FFADE

    Fade flags for CPlayerFuncs::ScreenFade and ScreenFadeAll

    Values

    Name Value Description
    FFADE_IN 0 Just here so we don't pass 0 into the function
    FFADE_OUT 1 Fade out (not in)
    FFADE_MODULATE 2 Modulate (don't blend)
    FFADE_STAYOUT 4 ignores the duration, stays faded out until new ScreenFade message received
