    RELATIONSHIP

    Relationship between (monster) entities

    Values

    Name Value Description
    R_AL -2 Ally. Good alternative to R_NO when applicable.
    R_NO 0 No relationship. Disregard.
    R_DL 1 Dislike. Will attack.
    R_HT 2 Hate. Will attack this character instead of any DISLIKE'd characters.
    R_NM 3 Nemesis. A monster will ALWAYS attack its nemesis, no matter what.
    R_FR 4 Fear. Will attack.
